Plataformas


WepSIM funciona en cualquier plataforma que use alguno de los navegadores Web más populares:
Google Chrome 90+, Mozilla Firefox 92+, Microsoft Edge 95+, Apple Safari 15+

Una versión para Android 9.0+ está también disponible en la tienda de Google Play.

Tecnologías


WepSIM usa diferentes tecnologías como:

  • Frameworks: Bootstrap, Vue, Jquery
  • Iconos: Font Awesome, icon-icons
  • Plug-ins y extensiones: Bootbox, Editor CodeMirror, Biblioteca de visualización vis.js, Jquery Knob
