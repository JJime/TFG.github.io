Platforms


WepSIM works in any platform that uses any of the most popular modern Web Browsers:
Google Chrome 90+, Mozilla Firefox 92+, Microsoft Edge 95+, Apple Safari 15+

An Android 9.0+ version version is also available at Google Play.

Technologies


WepSIM uses different technologies such as:

  • Frameworks: Bootstrap, Vue, Jquery
  • Icons: Font Awesome, icon-icons
  • Plug-ins & extensions: Bootbox, CodeMirror Editor, vis.js visualization library, Jquery Knob
